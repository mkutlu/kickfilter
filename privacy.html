Privacy Policy

Kick Channel Filter is a client-side browser extension for Google Chrome.

Data Collection

This extension does not collect, transmit, or sell any personal or usage data.

Local Storage

Settings (e.g., your blocked channel list and toggle preferences) are stored using Chrome's chrome.storage APIs (storage.sync or storage.local) on your device or synced across your Chrome profile.

Network Requests

The extension does not send data to external servers. It runs only on https://kick.com/* pages to hide specified channels and detect spam-like names.

Permissions

  • storage — to store your settings.
  • tabs — used only to read the current tab URL when clicking “Add current”.
  • host_permissions: https://kick.com/* — to operate on Kick.com pages.

Contact

If you have questions or issues, please use the Support contact from the Chrome Web Store listing.
